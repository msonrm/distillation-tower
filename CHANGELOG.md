## [0.1.0] - Initial Prototype

### Added
- セル・オートマトンによる蒸留シミュレーション
  - イジングモデル + カワサキダイナミクス
  - 潜熱モデルによる相転移（気化・凝縮）
  - フーリエの法則による熱伝導
  - メトロポリス法による交換判定
- 100×100 グリッドの Canvas 描画
- リアルタイムパラメータ調整 UI
  - 物質A/B の物性値スライダー
  - 壁・空気のプロパティ
  - 6×6 相互作用マトリックスエディタ
  - システムパラメータ（重力、冷却係数、熱源/冷却温度）
- 温度ベースの色ブレンド表示
- FPS・物質量・平均温度のリアルタイム表示
